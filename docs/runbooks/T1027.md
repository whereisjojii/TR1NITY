# Obfuscated files or information (T1027)

Encoded, packed, or otherwise obfuscated payloads. Often a precursor
or a partner technique to T1059 / T1071.

## Triage

- Determine the obfuscation type: base64, hex, XOR, custom packers,
  steganography in images.
- Was this caught by AMSI / EDR or after-the-fact in logs?

## Investigation

- Decode and inspect the payload in a sandbox. Match strings against
  known YARA rules.
- Pull surrounding events: process creation, file writes, outbound
  connections.

## Containment

- Block the originating delivery channel (URL, mail, USB device).
- Scan adjacent hosts for the same payload hash.

## Eradication & lessons

- Tune AMSI / EDR signatures.
- Consider script-block / cmdline auditing if the obfuscation came
  from PowerShell.
