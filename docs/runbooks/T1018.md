# Remote system discovery (T1018)

Adversary enumerates other hosts on the network — `net view`,
`arp -a`, `ping sweep`, `nltest`, `BloodHound`. Often a precursor to
T1078 (Valid accounts) or T1046 (Network service scanning).

## Triage

- Confirm the source isn't a legitimate vulnerability scanner, asset
  inventory tool, or NAC agent.
- Pull the process command line — `net view /domain` from a
  workstation is often part of an early kill-chain phase.

## Investigation

- Was the source recently logged into from an unusual geography?
  T1078 follow-up.
- Did the discovery results land in an exfiltration channel later?

## Containment

- For low-confidence cases, monitor and tag the host. Discovery alone
  is rarely worth isolating, but it is worth raising the host's
  baseline alert priority for 24 h.

## Eradication & lessons

- If the host is confirmed compromised, follow the broader IR plan
  (memory snapshot, isolate, reimage).
