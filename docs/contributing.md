# Contributing

Full contributing guide: [`CONTRIBUTING.md`](https://github.com/whereisjojii/TR1NITY/blob/main/CONTRIBUTING.md) at the repo root.

## TL;DR

```bash
git clone https://github.com/whereisjojii/TR1NITY.git
cd TR1NITY
pip install pre-commit && pre-commit install
make up
```

- One module change per PR.
- Defensive security only.
- No paid APIs.
- No file > 10 MB committed.
- Pre-commit hooks must pass; CI must be green.
